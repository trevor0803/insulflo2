# Images

Drop your photos in this directory. Recommended file names so the site
picks them up automatically (the HTML already references some of these):

## Hero / homepage
- `hero.jpg` — main homepage hero photo (recommended 1600×2000, portrait)
- `crew.jpg` — Russ + son + son-in-law team photo (recommended 1200×1500)

## Service pages
- `spray-foam.jpg`
- `blown-in.jpg`
- `removal.jpg`
- `windows.jpg`
- `storm-damage.jpg`

## Portfolio (8 images)
- `portfolio-1.jpg` through `portfolio-8.jpg`
  (recommended 1200×900 each)

## General guidelines
- **Format**: JPEG for photos, PNG for graphics with transparency, SVG for logos/icons
- **Size**: aim for under 300KB per image — use [squoosh.app](https://squoosh.app) to compress
- **Dimensions**: photos look best at 1200px wide minimum
- **Alt text**: descriptive, mention the service & location when relevant

## How to swap a placeholder for a real photo

Once you drop a photo in this folder, see `IMAGE-GUIDE.md` in the project
root for exactly which HTML lines to edit. It's only a few lines per page.

The current placeholders are deliberately neutral so the site looks
intentional until photos arrive — not "broken" or "missing image."
